# BinoCart Brand Assets

This folder contains BinoCart visual assets organized for app use.

## Folder Structure

- `logos/` - horizontal, stacked, wordmark, and icon-only logo assets.
- `icons/app/` - app icon, Apple touch icon, Android icon, and favicon PNGs.
- `icons/variations/` - alternate Bino icon treatments.
- `icons/feature/` - Scan, Compare, Save, and Smarter feature icon strip.
- `mascot/` - Bino mascot head, expressions, and character poses.
- `brand-system/` - color palette and typography reference crops.
- `usage/` - social preview, website header, and mobile app preview examples.
- `reference/raw/` - original uploaded screenshot crops.
- `reference/crops/` - larger section crops preserved for context.
- `manifest.csv` - generated asset index mapping curated files to their raw source screenshots.

## Current Asset Quality

The usable assets in this folder were generated from screenshot crops. They have been organized, trimmed where useful, and exported as higher-resolution PNG files.

These are usable for prototype UI work, README visuals, internal planning, and low-risk app placeholders. They are not equivalent to production vector exports from the original design source.
